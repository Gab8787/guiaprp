Operadores lógicos

os operadores lógicos são utilizados em todas as linguagens, eles sao operadores importantes para montar seu código

(and) Retorna o true(verdadeiro) se for verdadeiro

(or) Retorna true(verdadeiro) se algumas das afirmações forem verdadeiras

(not) Ele nega e retorna false (falso) se o resultado for verdadeiro

Operadores de comparação

Sua utilidade é comparar dois valores com o retorno de falso ou verdadeiro(False ou True)

(>) Identifica se um valor é maior que o outro

(<) Identifica se um valor é menor que o outro

(==) Identifica se um valor é igual o outro

(!=) Identifica se um valor é diferente do outro

(>=) Identifica se um valor é maior ou igual ao outro

(<=) Identifica se um valor é menor ou igual ao outro

Resultado

4 > 8 = False

4 < 2 = True

4 == 4 = True

4 != 5 = True

5 >= 4 = True

5 <= 2 = False

manipulação de strings

aqui veremos algumas das principais utilidades de manipulações de string

len() retorna o tamanho da string

capitalize() retorna a string com a primeira letra maiúscula

count() informa qunatas vezes um caractere ou uma sequencia aparece na string

islower() verifica se todas as letras de uma string são minúsculas

isupper() verifica se todas as letras de uma string são maiúsculas

upper() retorna uma copia das strings retornando todas as letras para maiúsculas

lower() retorna uma copia das strings retornando todas as letras para minúsculas

title() converte para maiúsculo as primeiras letras de cada palavra na string
